# ts-tutorial 기본 세팅방법

        ```bash
            1. npm init
            script 부분 작성시 ts-node index.ts(파일명 확장자까지) 건너뛰기 가능
            npx ts-node index.ts 로 대체가능
            2. npx tsc --init //타입스크립트 config파일 생성

            3. npm i ts-node typescript

            4. 즐코딩

            5. npm test 또는 npx ts-node 테스트하고싶은파일.ts
        ```
